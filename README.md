# Diwali Sales Data Analysis (Python + Power BI)

## Project Overview

This project analyzes Diwali sales data to understand customer purchasing behavior and identify important business insights. The analysis was performed using Python for data processing and visualization, and Power BI for creating an interactive dashboard.

The goal of the project is to explore sales patterns based on customer demographics such as gender, age group, state, occupation, and product category.

---

# Tools and Technologies Used

## Python

Python was used for data analysis and visualization.

Libraries used:

* Pandas
* NumPy
* Matplotlib
* Seaborn

Environment:

* Jupyter Notebook

## Power BI

Power BI was used to create an interactive dashboard that visually represents the sales data and insights.

---

# Dataset

The dataset contains customer purchase information during the Diwali festival.

Important fields include:

* User_ID
* Cust_name
* Product_ID
* Gender
* Age Group
* Age
* Marital_Status
* State
* Zone
* Occupation
* Product_Category
* Orders
* Amount

Each row represents a transaction made by a customer.

---

# Python Data Analysis

The dataset was analyzed using Python in Jupyter Notebook.

Steps performed:

1. Imported the dataset using Pandas
2. Cleaned the dataset and handled missing values
3. Explored the data using descriptive statistics
4. Created visualizations using Matplotlib and Seaborn

Visualizations created:

* Sales by Gender
* Sales by Age Group
* Sales by State
* Sales by Occupation
* Product Category Distribution

These visualizations helped identify patterns in customer purchasing behavior.

---

# Power BI Dashboard

An interactive dashboard was created using Power BI to visualize the insights from the dataset.

The dashboard includes:

* KPI Card showing Total Diwali Sales
* Sales by Gender
* Sales by Age Group
* Sales by State
* Sales by Occupation
* Product Category Distribution
* Sales Trend Line Chart
* Interactive Filters (State and Gender)

These visuals allow users to explore the data dynamically.

# Key Insights

* Female customers contributed more to overall sales.
* The 26–35 age group generated the highest sales.
* Uttar Pradesh, Maharashtra, and Karnataka contributed significantly to total sales.
* IT and Healthcare professionals showed high purchasing activity.
* Food and Clothing categories dominated festive purchases.
